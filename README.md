#Concurrent TCP/IP Day Time Server - Client in C
**SAMPLE OUTPUT:**
-----------------
***Server side:***
> gcc filename.c -o filename 

> ./filename
```
text message here .. :hello
CLIENT: hi
```
***Client side:***
> gcc filename.c -o filename 

> ./filename 0.0.0.0
```
Localhost: 192.168.233.219
Local Port: 9002
Remote Host: 0.0.0.0

SERVER : hello
text message here... :hi
```
